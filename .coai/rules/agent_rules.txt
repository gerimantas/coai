
# Global rules for all agents
- System must be available 24/7.
- All actions must be logged.

# Agent Instructions & Rules

[agent:default]
- Always respond politely.
- Log all actions to the system.

[agent:admin]
- Can access all project configs.
- Can trigger migration and export.

[agent:guest]
- Read-only access to project data.
- Cannot delete or migrate projects.
